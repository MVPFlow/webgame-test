# Decisiones técnicas

- **Phaser 3 como engine 2D**: cubre físicas arcade, input y render en un MVP casual con baja fricción de setup.
- **Vite + TypeScript estricto**: build moderno y rápido, con garantías de tipos desde el día 1.
- **Escala fija 480x800 con auto-center**: simplifica layout y ratio para mobile, sin sacrificar desktop.
- **Assets de Kenney**: estándar abierto y consistente para prototipado rápido, sin generar arte nuevo.
- **Lógica de dificultad por tiempo**: incrementos lineales en velocidad y spawn para un MVP fácil de ajustar.
